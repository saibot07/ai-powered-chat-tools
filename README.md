# AI-Powered Chat Tools

This repository provides a Python tool that interfaces with OpenAI's ChatGPT API to implement AI-powered contextual chat features.

## Features
- Seamlessly integrates with OpenAI's ChatGPT API.
- Context-aware conversation management.
- Easily customizable for specific use cases.

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/yourname/ai-powered-chat-tools.git
   ```
2. Install required dependencies:
   ```bash
   pip install openai
   ```

## Usage
1. Replace `YOUR_API_KEY` in `main.py` with your OpenAI API key.
2. Run the script:
   ```bash
   python main.py
   ```
3. Input your message to start a conversation with the AI assistant.

## Customization
Modify the `system_prompt` variable in `main.py` to customize the assistant's behavior.
